# GCode-Sentinel — Project Instructions

## Commit and Push Messages

**NEVER** mention Claude Code, Claude, AI assistance, or any automated tool in commit messages, push messages, or PR descriptions. Only describe the contribution itself as the author's own work.

Do **NOT** add `Co-Authored-By` trailers or any attribution lines referencing AI tools.

Example of what NOT to write:
```
feat: add serde dependencies

Co-Authored-By: Claude Sonnet 4.6 <noreply@anthropic.com>
```

Example of correct format:
```
feat: add serde and serde_json dependencies for JSON report output
```
